## HospiSearch - Hospital Search Platform

**Duration:** Jan 2023 - May 2023

### Overview
HospiSearch is a comprehensive hospital search platform designed to help users find and compare healthcare facilities efficiently.

### Key Features
- **Command Line Interface:** Implemented 5 essential CLI commands: Undo, Redo, Light, Dark, and View
- **User Interface Design:** Led design and construction of User Interface, prioritizing intuitive navigation and user-centric design principles
- **Responsive Design:** Ensured optimal user experience across different devices and screen sizes

### Technologies Used
- Frontend: React.js, HTML, CSS
- Backend: Node.js, Express
- Database: MongoDB
- Tools: Git, VS Code

### Impact
- Delivered a fully functional hospital search platform
- Implemented intuitive user interface with focus on accessibility
- Created comprehensive CLI tools for power users
